Open with webarchive

Requirements

  • http-server

Opening

  1. Download the webHelpPWS-MVDH2-all.zip file.
  2. Unzip the webHelpPWS-MVDH2-all.zip file.
  3. Open the extracted files
  4. Run http-server.
  5. Got to http://localhost:8080/.

Open with MD files

Opening

  1. Download the mdHelpPWS-MVDH2-all.zip file.
  2. Unzip the mdHelpPWS-MVDH2-all.zip file.
  3. Open the MD files.

Open with WriterSide

Opening

  1. Clone the Dog-Breed-Finder repo.
  2. Open WriterSide in the main directory.
  3. Optionally open the preview in a web browser or export it.
